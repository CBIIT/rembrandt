About REMBRANDT Functions
Users can perform a variety of tasks in REMBRANDT. The following table describes each REMBRANDT task.
REMBRANDT user tasks
Task
Description
Perform Simple Searches by Gene ID and SNP Probeset ID
Search the database and view the following search results:
•
Gene Expression plots
•
Kaplan-Meier Survival plots
For more information, see Simple Search Overview)
Perform Advanced Adhoc Queries
Query the database and view search results for the following:
•
Gene Expression analysis
•
Copy Number Data analysis
•
Clinical Study analysis
For more information, see Advanced Searches Overview.
Perform High Order Analyses
Run higher order analyses, including class comparisons, hierarchical clustering and principal component analyses.
For more information, see High Order Analysis Overview.
Generate Reports
View Advanced Search and High Order Analysis results. Also download static, archive files for use in BRB-ArrayTools. For more information, see Results Overview.
Manage Lists
Manage user-defined or study-defined patient identifier, gene, or reporters lists. For more information, see Managing Lists).
