Welcome to REMBRANDT
The REMBRANDT workspace comprises a set of five tabs, a blue panel, help links, and a logout link. The five tabs enable you to perform the following functions:
1.
Perform a simple search (see Simple Search Overview)
2.
Create a complex queries (see Advanced Searches Overview)
3.
Perform higher order analyses (see High Order Analysis Overview)
4.
View results of searches (see Results Overview)
5.
Manage lists (see Managing Lists Overview)
The blue panel may appear blank when you start using REMBRANDT, but once you start using REMBRANDT, information from various functions displays here:
•
Queries defined with Advanced Search function
•
Any filter settings defined in the High Order Analysis function
•
List information managed in Manage Lists function
To log out of REMBRANDT, see Logging Out.
To obtain help about REMBRANDT, see Getting Help.
