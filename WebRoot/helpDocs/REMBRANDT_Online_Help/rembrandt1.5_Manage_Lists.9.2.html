Managing Lists Overview
The Rembrandt List Management function centralizes all activities pertaining to the creation and management of user-defined, as well as study-defined, PatientDID Lists, Gene Lists, and Reporters Lists. With these lists, you can further refine queries or facilitate analysis.
Note:
You can add a saved gene or IMAGE clone list to an advanced gene expression search. You can also add a saved gene list or SNP probeset list to an advanced copy number search.
The blue panel displays each list type and the associated lists. You can mouse-over a list and display the data items. Using the Manage List function, you can perform the following functions:
•
View data items in a list (see Viewing the Data Items in a List)
•
Create new lists from existing lists (see Removing Data Items to Create a New List and Combining Existing Lists to Create a New List)
•
Delete lists (see Deleting an Entire List)
•
Add lists by uploading them or typing them (see Adding a New “Custom” List)
