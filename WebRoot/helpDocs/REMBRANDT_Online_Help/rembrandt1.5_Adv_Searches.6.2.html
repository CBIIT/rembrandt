Advanced Searches Overview
The Advanced Search function enables you to create multiple searches and then group the searches into a single, complex query from which you can generate reports. The following is an overview of this process.
1.
The Advanced Search - Build Query page enables you to define advanced searches from three categories:
–
Gene Expression Analysis (see Gene Expression Advanced Search)
–
Copy Number Analysis (see Copy Number Advanced Search)
–
Clinical Analysis (see Clinical Study Advanced Search)
2.
Once you create a search, the Advanced Search - Build Query page appears. You can add more searches or copy, edit and delete existing searches (see Managing Advanced Searches).
3.
Once the searches are complete, click the Finalize Query button or Refine Query option.
4.
On the Refine Query page (see Refining a Query), group the searches with parentheses and AND/OR conditional items to create one complex query.
5.
Validate the query and run the report. Report results are listed on the View Results page (see Results Overview).
