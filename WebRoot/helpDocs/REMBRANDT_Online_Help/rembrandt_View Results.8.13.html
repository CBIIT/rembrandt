Copy Number Sample Report
The Copy Number Data Per Sample report displays the copy number data from Affymetrix 100K SNP arrays. The CHP files from the Affymetrix Gene Chip Operating System were processed using the Affymetrix GDAS (GeneChipÂ® DNA Analysis Software). Copy number data was collected for each mapping SNP reporter on the Chip, for all the tumor samples. Each column represents a sample, and the samples are grouped based on the tumor sub-type.
