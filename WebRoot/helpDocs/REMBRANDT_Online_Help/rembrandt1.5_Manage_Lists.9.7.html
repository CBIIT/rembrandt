Creating New Custom Lists from Existing Lists
You may create new lists from existing lists:
•
Join combines two or more lists.
•
Intersect creates a new list of intersecting items (items that appear on more than one list).
•
Different creates a new list of non-intersecting items.
To create a custom list from existing lists follow these steps:
1.
Select the list type from the menu beneath the function tabs at the top of the page. A list for that category appears.
2.
Select the lists you would like to work with by clicking the box beside each list name.
3.
Enter a name for the new list you are creating, and then click the appropriate button: Join, Intersect, Difference. The new list will appear in red in the appropriate category. (Figure: Create Custom Lists by Selecting Join, Intersect, Difference)
4.
To open and view the newly created list or save it to your computer, click on the list name in the right-hand column). Click Open or Save.
