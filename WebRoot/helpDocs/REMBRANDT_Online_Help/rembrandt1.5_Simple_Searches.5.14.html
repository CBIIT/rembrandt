Redrawing the K-M Survival Plot for Copy Number Data
To redraw a KM Copy Number graph, follow these steps:
Note:
If you restricted the search to a group, Constrained to group appears at the top.
1.
Select the amplification and deletion criteria.
2.
To visualize the K-M plot for the unified probeset, select a value from the Reporters drop-down list.
3.
Click the Redraw Graph button.
For more information, see Understanding K-M Survival Plot for Copy Number Data.
