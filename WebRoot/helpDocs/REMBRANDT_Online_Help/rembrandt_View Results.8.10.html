Differentiating Data (Show All Values Toolbar)
To differentiate between missing values in the array and data that did not meet your search criteria, follow these steps:
1.
Click Show All Values on this Report on the Show all Values toolbar.
The samples that did not meet your criteria, appear in gray. A value of Null indicates a missing value for that reporter.
2.
To display the previous report before you clicked the Show All Values on this Report button, click the View Previous Report button.
