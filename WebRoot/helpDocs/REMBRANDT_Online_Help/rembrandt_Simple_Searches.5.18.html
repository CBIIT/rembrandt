Conducting a Simple Search > K-M Sample Search
K-M Sample Search
To create a Kaplan-Meier survival plot for sample data (K-M Sample), follow these steps:
1.
From the Simple Search tab, select Create Kaplan-Meier survival plot for Sample Data.
2.
Select a sample from each drop-down list for comparison purposes.
3.
Click the Go button. The Kaplan-Meier survival plot appears.
For information about the Gene Expression Plot, see Understanding K-M Survival Plot for Sample Data.
