REMBRANDT’s Opening Page
REMBRANDT’s opening page enables you to perform the following tasks:
•
Register new users and log in current users with the Browse REMBRANDT Data button.
•
Download an online version of the REMBRANDT User’s Guide with the Download User Guide (PDF) link.
•
Run REMBRANDT tutorials that will assist you in using new REMBRANDT options with the View Tutorials link.
•
Provide the REMBRANDT team with any feedback about the product with the Provide us your feedback link.
For additional help using REMBRANDT, see Getting Help.
