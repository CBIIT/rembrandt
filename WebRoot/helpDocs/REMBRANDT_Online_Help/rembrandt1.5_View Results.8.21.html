Downloading BRB Files
REMBRANDT enables you to analyze data using BRB-ArrayTools created by the Briometric Research Branch of the National Cancer Institute. BRB-ArrayTools is a comprehensive state-of-the-art statistical analysis system for microarray gene expression data. It is free for non-commercial purposes and can be licenced for commercial purposes from the NIH. BRB-ArrayTools installs as an Excel plug-in using a self-installer. To download BRB-ArrayTools, go to:
http://linus.nci.nih.gov/~brb/BRB-ArrayTools.html
On the View Results page, REMBRANDT provides a link to the static BRB archive files which enable you to download the appropriate files based on user role as follows:
•
Public users receive the Public dataset file (Rembrandt-Brain-Public-Project.zip)
•
Institution users receive the institution data and public dataset files.
•
Super users receive all dataset files.
To analyze a dataset, follow these steps:
1.
Download and install BRB-ArrayTools.
2.
Click the file name on the View Results page.
3.
Unzip the REMBRANDT static BRB archive file(s).
4.
Open the project worksheet in Excel on a Microsoft Windows PC.
