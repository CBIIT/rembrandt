Importing Lists and Queries
To import a list or a query that you have saved in XML format at an external location, follow these steps:
1.
Select the My Workspace tab.
On the page that opens, Rembrandt displays the following sub-categories as hyptertext links: Manage Lists, Organize, Import, and Export. By default, the page opens to the Manage Lists subcategory.
2.
At the top of the Manage List page, click the Import hypertext link.
The Import Workspace text describes how you can import lists and queries.
3.
At the bottom of the page, select the file type you are importing, Query or List.
4.
Click the Browse button to navigate for the file(s).
5.
Click the Import button.
Rembrandt informs you if the file you are trying to import does not comply with acceptable format standards. If it does meet requirements, the system opens the Organize Workspace page where you will see the imported file displaying in a new “Imported” folder named with the date and time stampIf a file with an identical name already exists in Rembrandt, the program renames the imported file.
You can import only one file at a time; import of a “batch” of files is not possible.
