The following Advanced Search reports are generated:

•

View Results displays the query name and the output generated for the query. To view the report, click the report name and the file opens in a new window.
All Advanced Search options (Gene Expression, Copy Number Data, and Clinical) generate a Clinical report. For more information about Clinical Reports, see Clinical Reports.
