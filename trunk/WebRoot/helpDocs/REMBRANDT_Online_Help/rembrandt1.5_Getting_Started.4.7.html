Getting Help
Information about how to use REMBRANDT is easily accessed from REMBRANDT’s menu in the top left of the REMBRANDT workspace.
REMBRANDTThe following table describes each item on the REMBRANDT menu.
Getting help with REMBRANDT
Help
How to Access
Complete online help
To access the complete version of online REMBRANDT help, click the help linklocated in the REMBRANDT menu.
For complete page-level help, click on any REMBRANDT page.
For brief field help, click For help on using REMBRANDT online help, see Using REMBRANDT Online Help.
Application support
To obtain support for REMBRANDT, click the support link located in the REMBRANDT menu.
Tutorials
To access REMBRANDT tutorials, click the tutorials link located in the REMBRANDT menu.
User’s Guide
To access a pdf version of the REMBRANDT User’s Guide, click the user guide link located in the REMBRANDT menu.
